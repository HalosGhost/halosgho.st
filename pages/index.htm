Sam Stuewe (halosghost)

Background

Though most of my personal projects are end-user facing, almost all my personal and professional programming experience is in lower-level languages (primarily C11). I also have worked with several higher-level languages both personally and professionally (including Lua, Haskell, C#, Python, Java and several others).

I am a daily user of several linux distributions, and I have professional experience doing day-to-day systems administration and deployment of Linux, Windows and macOS physical and virtual machines.

Projects

Though shaman and pbpst are fairly stable, none of these projects are complete; there are still many more things I would like to implement in all of them.

  • shaman, a simple CLI weather program
  • pbpst, a simple, but featureful pastebin client for pb deployments
  • adarcroom, a C/ncurses port of A Dark Room
  • .bin, a set of small, helpful utilities
  • .dotfiles, my Linux userspace configurations
